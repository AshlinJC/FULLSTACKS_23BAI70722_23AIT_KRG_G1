# Project Management App (Trello-like) - MongoDB Version

This scaffold sets up a Trello-style task management app with:
- Frontend: React (Vite)
- Backend: Node.js + Express + Socket.IO
- Database: MongoDB (via Mongoose)
- Auth: JWT
- Real-time updates: WebSocket (Socket.IO)

## Quick Start

### Backend
```bash
cd backend
npm install
# set env variables: JWT_SECRET and MONGO_URI
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Then open http://localhost:5173
